# Cypher Sheet 🎲

  Ficha de personagem digital e editável para o sistema de RPG de mesa **Cypher System**.

  ## Funcionalidades

  - Atributos do personagem (Força, Velocidade e Inteligência)
  - Seleção de cifras e equipamentos
  - Campo de anotações
  - Geração de PDF da ficha pronta para impressão

  ## Tecnologias

  - React
  - Context API
  - CSS
  - JavaScript

  ## Como rodar

  ```bash
  # Entre na pasta do projeto
  cd app

  # Instale as dependências
  npm install

  # Rode em desenvolvimento
  npm start
  ```

  Acesse em: `http://localhost:3000`
